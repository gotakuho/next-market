####マークダウン（Markdown）完全ガイド

マークダウン（Markdown）は、プレーンテキストを用いて、構造化された文書を素早く作成するための軽量マークアップ言語です。

## 1. マークダウンのメリット

* **学習コストが低い**: 記号を覚えるだけで、誰でもすぐに使い始められます。
* **執筆に集中できる**: マウスを使ってメニューから装飾を選ぶ必要がなく、タイピングに専念できます。
* **汎用性が高い**: 多くのサービス（GitHub, Notion, Qiita, Slackなど）で採用されており、PDFやHTMLへの変換も容易です。

## 2. 基本的な記法

### 見出し (Headings)
`#` の後に半角スペースを入れます。

```
# 見出し1
## 見出し2
### 見出し3
```

### リスト (Lists)
箇条書きは `-`、`+`、`*` のいずれか、番号付きは `数字.` を使用します。

```
- 箇条書きアイテム1
  - ネスト（インデント）
- 箇条書きアイテム2

1. 番号付きアイテム1
2. 番号付きアイテム2
```

### 強調 (Emphasis)
文字を囲みます。

```
**太字**
*斜体*
~~打ち消し線~~
```

### リンクと画像 (Links & Images)
括弧を組み合わせて記述します。

```
[Googleへのリンク](https://www.google.com)
![画像の代替テキスト](画像のURL)
```

### 引用 (Blockquotes)
行頭に `>` を置きます。

```
> 引用文です。
> 複数行にわたる場合も各行に記述します。
```

### コード (Code)
短いコードはバッククォート ` で囲み、ブロック単位は ``` で囲みます。

### 表 (Tables)
パイプ `|` とハイフン `-` を使って作成します。

| 項目 | 内容 |
| :--- | :--- |
| 言語 | Markdown |
| 用途 | 文書作成 |

## 3. まとめ
マークダウンは、シンプルでありながら表現力豊かなドキュメント作成を可能にします。まずは見出しとリストから使い始めてみてください。
